# prog3-a1-poo-ThiagoVenturini

Thiago Henrique Venturini, 5º Semestre Ciências da Computação Campus São Miguel do Oeste,
Esse é um projeto realizado em PHP puro, visando conceitos como POO, foi realizado um cadastro de Usuários com nome, email, senha

Instruções para simular localmente: Para simular localmente, primeiramente, é necessário clonar o projeto para a sua máquina, depois disso e so abrir o Chrome ou navegador de preferencia e informar o diretório que foi clonado o projeto

Exemplo: D:\prog3-a1-poo-ThiagoVenturini, após fazer isso e so adicionar \index.php para acessar a página inicial do cadastro 

**Importante**: É necessário ter alguma ferramenta como XAMPP ou similares para conseguir acessar o projeto
